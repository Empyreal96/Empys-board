Create Custom Windows Phone Packages

This guide will explain the basics on how to create the XML definition files which include files and registry entries, as well as actually packaging the archive and deploying to your device

Also a small guide on creating provisioning packages for UWP Apps to add to a custom package, to allow apps to be installed like a "System app"

These guides are hosted on GitHub

Creating Packages

Using tools provided in the repo, this will let you learn how to create the packages, there are limits, for example: Files and Registry keys that collide with already existing items in other Packages will cause failure when deploying. Also Unlocked Bootloader is needed to deploy these (Relock can be done after deployed successfully)
